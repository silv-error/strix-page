Strix is built with a privacy-first mindset. We collect only what is necessary to operate — nothing more. We do not sell, trade, or share your data with third parties.

01

Overview

This Privacy Policy describes how Strix ("the Bot," "we," "us") collects, uses, and protects information when you use our Discord bot service. By using Strix, you agree to the data practices described in this policy.

Strix is hosted on Google Cloud and interacts with Discord's API to provide automated member verification management.

02

What We Collect

We collect only what is strictly necessary for the Bot to function:

Data Purpose Stored?
Server (Guild) ID Identify which settings belong to which server Yes
Kick timer & role name Apply correct settings per server Yes
Log channel ID Know where to send log messages Yes
Member join timestamp Calculate if timeout has elapsed Temporary
Member ID (Snowflake) Track unverified members; cleared on verify or kick Temporary
03

What We Do NOT Collect

We explicitly do not collect or store:

  • Message content of any kind
  • Usernames, display names, or avatars
  • Email addresses or any contact information
  • IP addresses of users or server members
  • Voice channel activity
  • DM content or private conversations
  • Data from channels the Bot is not configured to access

Strix does not have the MESSAGE CONTENT intent enabled. It cannot read the content of any messages in your server.

04

How We Use Data

All collected data is used exclusively to operate the Bot's core features:

  • Server configuration — apply the correct timeout and log channel settings per server
  • Verification tracking — temporarily monitor which members are unverified and for how long
  • Automated moderation — kick unverified members once their timer elapses, per admin configuration
  • Logging — send formatted log entries to the designated channel when actions are taken

We do not use your data for analytics, advertising, profiling, or any purpose beyond operating the Bot.

05

Data Storage & Security

Strix is hosted on Google Cloud, which provides enterprise-grade infrastructure security. Data is stored in a secured database accessible only to the Bot's runtime environment.

Security measures include restricted access controls, no public exposure of raw data, and secure handling of all credentials. However, no method of storage is 100% secure — we commit to industry-standard best practices.

06

Data Retention

  • Unverified member records — deleted immediately upon verification or kick
  • Server configuration — retained as long as the Bot is in your server
  • After Bot removal — all data deleted within 30 days
07

Data Sharing

We do not sell, rent, or share your data with any third parties. The only external services that receive data are:

  • Discord — as required to interact with their API. Governed by Discord's Privacy Policy.
  • Google Cloud — as our hosting provider. Governed by Google's Privacy Policy.

We may disclose data if required by law, but will notify affected parties where permitted.

08

Your Rights

As a server administrator, you have the following rights:

  • Access — request information about what data we hold for your server
  • Deletion — request deletion by contacting us or removing the Bot
  • Correction — update configuration data anytime via slash commands
  • Opt-out — remove the Bot at any time, immediately stopping all data collection
09

Children's Privacy

Strix does not knowingly collect personal information from children under 13. If you believe a child has provided data without parental consent, contact us and we will delete it. All users must comply with Discord's minimum age requirements.

10

Changes to This Policy

We may update this Privacy Policy periodically. When we do, we will update the date at the top. Continued use of Strix after changes constitutes acceptance of the updated policy.

11

Contact

For questions, concerns, or data requests, reach out via the Bot's official Discord support server or by submitting a GitHub issue. We aim to respond to all privacy-related inquiries within 5 business days.
