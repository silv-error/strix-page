02

Description of Service

Strix is a Discord bot designed to help server administrators manage member verification. The Bot provides:

  • Automatic tracking of unverified members upon joining a server
  • Configurable verification role name and kick timer via /setup
  • Real-time tracking of unverified members and time remaining via /status
  • Dedicated log channel for kick events via /setlogchannel
  • Optional DM warnings to unverified members via /toggledm
  • Built-in help reference via /help
