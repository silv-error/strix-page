Features

Everything you need.

Configurable Timer

Set exactly how long unverified members have before being kicked. Fully adjustable per server.

Log Channel

Every join, warning, and kick is logged to your chosen channel with timestamps and context.

Slash Commands

Slash Commands

Simple to set up.

/setup

Configure the bot

Set the verification role name and kick timer. Run this first after adding Strix.

/status

View tracked members

See all unverified members currently being tracked and their remaining time before kick.

/setlogchannel

Set log channel

Choose which channel receives all kick logs and verification event entries.

/toggledm

Toggle DM warnings

Enable or disable DM warnings sent to unverified members before they are kicked.

/help

Show help

Display all available commands and a quick-start guide directly inside Discord.
